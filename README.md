## Deployment to GitHub Pages

1. Create a repo named `jidapa21.github.io` on GitHub
2. Upload all these files to the repo root
3. Go to **Settings → Pages → Source: main branch / root**
4. Your site will be live at `https://jidapa21.github.io`

## Adding Images

Add your project images to an `images/` folder:
- `images/object-detection.jpg` — Screenshot or photo of detection system
- `images/dashboard.jpg` — Screenshot of Grafana dashboard
- `images/workorder.jpg` — Screenshot of work order app
- `images/coop.jpg` — Screenshot of co-op platform
- `images/datamining.jpg` — Charts/visualizations from data mining

Recommended image size: **1200×675px** (16:9 ratio), JPEG under 200KB each.

If images are missing, beautiful placeholder visuals will display automatically.
