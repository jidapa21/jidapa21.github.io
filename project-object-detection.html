ML Object Detection System

A real-time object detection pipeline built for industrial quality assurance — identifying defects and anomalies on manufacturing lines with high confidence.

Overview

Project Overview

This project delivers a machine learning–powered object detection system designed for real-time quality assurance in manufacturing environments. Deployed during my internship at The Nawaloha Industry Co., Ltd., the system identifies product defects, misalignments, and anomalies directly on the production line.

The pipeline integrates computer vision models with industrial camera hardware, enabling automated visual inspection that replaces time-consuming manual checks.

Problem Statement

The Problem

Manual quality inspection in manufacturing is slow, inconsistent, and prone to human error — particularly for high-throughput production lines where inspectors must evaluate hundreds of components per hour. Missed defects result in costly downstream failures and returns.

The existing inspection process lacked automation and real-time feedback, causing bottlenecks and quality inconsistencies across production batches.

Solution Approach

How I Solved It

I implemented a trained object detection model (based on YOLO architecture) tuned specifically for the factory's defect categories. Key engineering work involved:

  • Collecting and annotating a dataset of manufacturing defect images specific to the production environment
  • Training and fine-tuning the detection model with transfer learning from pre-trained weights
  • Iteratively tuning confidence thresholds to minimize false positives while catching critical defects
  • Building a real-time inference pipeline integrated with industrial cameras via Python
  • Conducting systematic AI validation to confirm model reliability before production deployment
Key Features

Key Features

  • Real-time object detection running on live camera feeds from the production floor
  • Configurable confidence thresholds for different defect severity levels
  • Automated flagging and logging of detected anomalies with timestamps
  • Multi-class detection covering multiple defect categories simultaneously
  • Visual overlay output for operator review and manual override capability
  • Performance metrics dashboard showing detection accuracy over time
Result / Impact

Results & Impact

The deployed system achieved significant improvement in defect detection consistency over manual inspection. Automated alerts reduced the time between defect occurrence and correction. The confidence threshold tuning process became a reusable methodology for future model deployments at the facility. Validation procedures were documented and adopted as a standard QA protocol for AI-assisted inspection at the company.
